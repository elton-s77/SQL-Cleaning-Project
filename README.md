# SQL-Cleaning-Project

This project demonstrates a complete data cleaning process in SQL (MySQL) applied to a table called limpieza. The goal is to transform raw, inconsistent data into a clean dataset ready for analysis and dashboards.

🚀 Main Steps
Inspect initial records.

Create a stored procedure for validation.

Rename columns with encoding errors.

Detect and remove duplicates.

Trim extra spaces and normalize text.

Standardize categorical values (gender, contract type).

Convert salary from text to numeric.

Reformat dates and calculate age.

Prepare final dataset for export and reporting.

🛠️ Technologies
SQL (MySQL)

Text and date functions

Stored procedures

📊 Outcome
A clean, consistent table with correct data types, normalized values, and ready-to-use metrics for business intelligence tools like Power BI or Excel.
